# 链家网新房数据可视化及预测系统 python710

## 项目概述


## 技术栈

### 前端

- HTML/CSS/JavaScript
- Echarts：用于图表绘制

### 后端

- Python语言
- Flask框架：处理HTTP请求，提供API接口

### 数据库

- MySQL数据库
- SQLAlchemy ORM：简化数据库操作

### 爬虫

- requests库：用于数据爬取

### 机器学习

- scikit-learn库：进行数据预测分析

## 功能模块

1. 数据爬取模块：负责爬取链家网新房数据。
   - 使用`requests`库进行数据抓取。
2. 数据库模块：负责数据的存储和管理。
   - 使用MySQL数据库。
3. 后端模块：处理前端请求，提供数据接口。
   - 使用Flask框架。
   - 采用scikit-learn库的多元线性回归算法。
5. 可视化模块：将数据以图表的形式展示给用户。
   - 集成Echarts库。

## 系统角色

- 用户：查看数据预测和分析结果。
- 系统管理员：维护系统正常运行，管理数据。

## 运行环境

- Python环境
- Flask框架
- MySQL数据库
- Echarts库

## 部署步骤

1. 环境准备：搭建Python、Flask和MySQL环境。
2. 数据库配置：建立数据库和表结构。
3. 爬虫执行：定时爬取链家网新房数据。
4. 系统部署：部署Flask应用。

## 目录结构

```
/
├── app/                # Flask应用目录
│   ├── models.py       # 数据模型
│   ├── views.py        # 视图逻辑
│   ├── static/         # 静态文件目录
│   └── templates/      # 模板文件目录
├── database/           # 数据库相关脚本
├── scripts/            # 爬虫脚本
└── venv/               # Python虚拟环境
```

## 核心亮点

- 基于Flask框架的轻量级后端服务。
- 利用SQLAlchemy ORM简化数据库操作。
- 使用scikit-learn库进行有效的数据预测。
- 借助Echarts实现直观的数据可视化。
